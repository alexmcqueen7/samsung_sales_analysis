# Samsung Sales Analysis

## Description
This project uses a Samsung Global Product Sales Dataset, which simulates a realistic global sales ledger for Samsung Electronics products. The data includes four years of global Samsung product sales across 52 countries. The project uses Google BigQuery to analyze the data and Power BI to create an executive summary dashboard showcasing key data and findings from the analysis. The goal of this project was answer a set of self-determined business questions

## Questions Answered
1. Which regions and countries generate the most revenue, and how is this changing over time?
2. Which products and product categories are driving the highest revenue?
3. How do discounts affect revenue and units sold across regions and product categories?
4. Which sales channels and customer segments are most profitable, and how do demographics affect purchasing behavior?
5. How is 5G adoption impacting sales revenue and units sold across regions?
